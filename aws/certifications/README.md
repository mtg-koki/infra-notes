# AWS 認定資格

AWS認定資格の学習メモ・進捗管理用ディレクトリ。試験は定期的に改定されるため、各ファイルには学習時点のバージョン・受験日を明記する。

## 資格一覧

| ファイル | 資格 | レベル |
|---|---|---|
| [saa.md](saa.md) | AWS Certified Solutions Architect – Associate (SAA) | アソシエイト |
| [dva.md](dva.md) | AWS Certified Developer – Associate (DVA) | アソシエイト |
| [soa.md](soa.md) | AWS Certified SysOps Administrator – Associate (SOA) | アソシエイト |
| [sap.md](sap.md) | AWS Certified Solutions Architect – Professional (SAP) | プロフェッショナル |

## 各ファイルの構成

- 資格概要（試験コード、費用、有効期限）
- 出題範囲（Exam Domains）
- 学習リソース
- 学習計画・進捗
- 頻出トピック・要点整理
- 受験結果・振り返り

## 学習の進め方の指針

- アソシエイト資格は実務経験がある領域から着手し、経験の薄い領域（例: 開発者ならインフラ運用、インフラ担当なら開発）を重点的に補強する。
- プロフェッショナル資格はアソシエイト2つ以上の合格後、複合的なシナリオ問題への対応力を養ってから挑む。
