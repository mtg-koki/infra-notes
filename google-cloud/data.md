# Google Cloud データ分析基盤

## 概要

Google Cloudが強みとするデータ分析・データベースサービス群。BigQueryを中心としたサーバーレス分析基盤と、Pub/Sub・Dataflowによるストリーミング処理を含めて扱う。

## 主要サービス

- BigQuery（サーバーレスDWH、SQL分析）
- Pub/Sub（メッセージング、非同期イベント連携）
- Dataflow（ストリーミング/バッチデータ処理、Apache Beam）
- Cloud SQL / Cloud Spanner（リレーショナルDB、Spannerはグローバル分散DB）
- Bigtable（大規模NoSQL、時系列/IoTデータ向け）

## 設計ポイント

- BigQueryはパーティショニング・クラスタリングでスキャン量を削減し、クエリコストを最適化する。
- グローバルな強整合性が必要な大規模OLTPワークロードはCloud Spannerを検討する（コストは高い）。
- Pub/Sub + Dataflowでイベント駆動のストリーミングパイプラインを構築し、BigQueryへのリアルタイム連携を実現する。

## 運用ポイント

- BigQueryのスロット/オンデマンド課金モデルを利用パターンに応じて選択し、コストを監視する。
- データガバナンス（Data Catalog、列レベルセキュリティ、行レベルセキュリティ）を整備する。
- パイプラインの障害時リプレイ・データ品質チェックの仕組みを組み込む。
