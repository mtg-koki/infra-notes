# Professional Cloud Architect (PCA)

## 資格概要

- 試験形式: 選択式、ケーススタディ（架空企業のシナリオ）ベースの出題が中心
- 対象者: Google Cloud上でのアーキテクチャ設計経験がある層
- 問題数・試験時間・費用・合格ライン: 要検証（公式サイトで最新情報を確認）
- 有効期限: 3年
- 前提知識: ACE相当、または実務でのアーキテクチャ設計経験

- ソリューションインフラの設計・計画
- ソリューションインフラの管理・提供
- 技術的要件の分析と設計への反映
- セキュリティとコンプライアンスへの対応
- 技術的プロセス運用の最適化
- ビジネスの成功に向けたマネージド運用
