# Google Cloud 認定資格

Google Cloud認定資格の学習メモ・進捗管理用ディレクトリ。試験は定期的に改定されるため、各ファイルには学習時点の受験日・出題範囲バージョンを明記する。

## 資格一覧

| ファイル | 資格 | レベル |
|---|---|---|
| [ace.md](ace.md) | Associate Cloud Engineer (ACE) | Associate |
| [pca.md](pca.md) | Professional Cloud Architect (PCA) | Professional |
| [pde.md](pde.md) | Professional Data Engineer (PDE) | Professional |
| [pcse.md](pcse.md) | Professional Cloud Security Engineer (PCSE) | Professional |

## 各ファイルの構成

- 資格概要（試験形式、費用、有効期限、前提知識）
- 出題範囲
- 学習リソース
- 学習計画・進捗
- 頻出トピック・要点整理
- 受験結果・振り返り

## 学習の進め方の指針

- Associate Cloud Engineerは運用・構築の基礎を問う内容。実務でgcloud/コンソール操作に慣れてから受験すると効率的。
- Professional Cloud Architectはケーススタディ（架空企業のシナリオ）を軸にした出題が特徴。事前にケーススタディを読み込んでおく。
- Professional Data Engineer / Cloud Security Engineerは専門領域資格のため、担当業務に応じて優先度を決める。
