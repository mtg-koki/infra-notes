# Professional Cloud Security Engineer (PCSE)

- 試験形式: 選択式、ケーススタディを含む
- 対象者: Google Cloud上でのセキュリティ設計・実装経験がある層
- 問題数・試験時間・費用・合格ライン: 要検証（公式サイトで最新情報を確認）
