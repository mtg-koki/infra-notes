# Microsoft Certified: Azure Administrator Associate (AZ-104)

## 資格概要

- 試験コード: AZ-104
- 対象者: Azureリソースの日常的な運用・管理経験がある層
- 出題形式: 選択式、シナリオベースの問題、ラボ形式を含む場合あり
- 問題数・試験時間・費用・合格ライン: 要検証（公式サイトで最新情報を確認）
