# Azure 認定資格

Microsoft Azure認定資格の学習メモ・進捗管理用ディレクトリ。試験は定期的に改定されるため、各ファイルには学習時点の受験日・出題範囲バージョンを明記する。

## 資格一覧

| ファイル | 資格 | レベル |
|---|---|---|
| [az-900.md](az-900.md) | Microsoft Certified: Azure Fundamentals (AZ-900) | Fundamentals |
| [az-104.md](az-104.md) | Microsoft Certified: Azure Administrator Associate (AZ-104) | Associate |
| [az-305.md](az-305.md) | Microsoft Certified: Azure Solutions Architect Expert (AZ-305) | Expert |
| [az-400.md](az-400.md) | Microsoft Certified: DevOps Engineer Expert (AZ-400) | Expert |

## 各ファイルの構成

- 資格概要（試験コード、費用、有効期限、前提資格）
- 出題範囲（Skills Measured）
- 学習リソース
- 学習計画・進捗
- 頻出トピック・要点整理
- 受験結果・振り返り

## 学習の進め方の指針

- AZ-900は非エンジニア含め全体像把握用。実務者はAZ-104から着手しても問題ない。
- AZ-305はAZ-104合格（または同等知識）を前提に、アーキテクチャ設計の判断軸を問う内容が中心。
- AZ-400はAZ-104 + Azure Developer Associate相当の知識を前提とする複合資格。
